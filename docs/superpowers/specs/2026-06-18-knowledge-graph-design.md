**日期：** 2026-06-18
**版本：** 1.0
**状态：** 已批准

---

## 一、目标与愿景

将产业学院课程平台升级为**数据驱动的沉浸式探索系统**，帮助企业/访客快速发现课程资源，通过可视化关系图谱展现学院实力。

### 核心价值

- **数据驱动** — 用数据讲故事，展现学院课程分布与关联
- **沉浸体验** — 让用户愿意"逛"平台，通过交互探索发现
- **目标导向** — 快速定位所需课程，支持多维度筛选

### 目标用户

- **企业** — 了解学院实力、寻找合作课程
- **访客** — 探索课程资源、规划学习路径

## 三、核心模块设计

### 模块1：交互式关系图谱

#### 节点设计

| 节点类型 | 大小 | 颜色 | 标签 |
|----------|------|------|------|
| 学院 | 大（半径40px） | 按学院分配一组配色 | 学院名称 |
| 专业 | 中（半径25px） | 继承所属学院颜色 | 专业名称 |
| 课程 | 小（半径12px） | 继承所属学院颜色 | 课程名称（悬浮显示）|

#### 节点配色方案

```
数字技术学院：#1a73e8 (蓝色系)
人工智能学院：#8b5cf6 (紫色系)
网络安全学院：#10b981 (绿色系)
数字经济学院：#f59e0b (橙色系)
低空经济学院：#06b6d4 (青色系)
智能制造学院：#ef4444 (红色系)
智慧能源学院：#eab308 (黄色系)
智慧文旅学院：#ec4899 (粉色系)
智慧康养学院：#14b8a6 (青绿色系)
```

#### 交互行为

| 操作 | 行为 |
|------|------|
| 点击节点 | 高亮该节点及其直接关联节点，弱化其他节点 |
| 双击学院节点 | 进入学院详情视图 |
| 双击课程节点 | 打开课程详情弹窗 |
| 拖拽节点 | 自由拖拽，其他节点自适应 |
| 滚轮缩放 | 缩放整个图谱（0.3x - 3x）|
| 拖拽画布 | 平移整个视图 |
| 悬浮节点 | 显示tooltip，显示完整名称和基本信息 |

#### 力导向配置

```javascript
{
  forceCharge: -300,        // 节点斥力
  linkDistance: 100,        // 连接线长度
  collisionRadius: 50,      // 碰撞半径
  centerForce: 0.05,       // 向心力
  alphaDecay: 0.02,        // 衰减速度
}
```

### 模块2：可视化筛选器

#### 筛选维度

| 维度 | 选项 | 交互方式 |
|------|------|----------|
| 学院 | 多选，默认全选 | 复选框 + 颜色标签 |
| 课程类型 | 必修/选修/实践 | 多选按钮 |
| 关键词 | 输入框 | 实时搜索，高亮匹配节点 |

#### 筛选效果

- 隐藏不匹配的节点
- 相关节点重新布局
- 统计面板实时更新
- 显示"已筛选"遮罩效果

## 四、数据流设计

```
用户操作 → 筛选条件变更
    ↓
JavaScript 过滤数据
    ↓
D3.js 重新计算力导图布局
    ↓
动画过渡到新状态（300ms ease-out）
    ↓
更新统计面板
```

### 性能优化

- 使用 `requestAnimationFrame` 进行动画
- 数据量 < 200 节点时实时渲染
- 数据量大时使用虚拟化
- 筛选操作使用 `throttle`（200ms）
